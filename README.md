# 📈 MRF Stock Analysis & Live Dashboard

A complete stock market analysis and interactive dashboard project for **MRF Ltd. (MRF.NS)** using Python.  
This project performs financial data analysis, visualization, volatility measurement, return calculations, and interactive dashboard filtering for stock market insights.

---

## 🚀 Project Overview

This project helps users analyze the historical stock performance of **MRF Ltd.** through:

- Stock Price Analysis
- Moving Average Analysis
- Monthly Return Analysis
- Volatility & Risk Measurement
- Trading Volume Insights
- ROI / Investment Return Calculator
- Interactive Dashboard Filters
- Candlestick Charts
- Data Visualization Dashboard

The project is built using **Python, Pandas, Matplotlib, mplfinance, and ipywidgets**.

---

## 📂 Project Files

### 1. `MRF_Stock_Analysis.ipynb`
Jupyter Notebook containing:
- Data preprocessing
- Financial analysis
- Statistical calculations
- Visualizations
- Investment return analysis

### 2. `MRF_stock_analysis_LIVE_Dashboard.py`
Python dashboard script containing:
- Interactive dashboard
- KPI cards
- Dynamic filters
- Real-time chart updates
- User-controlled analysis
